AURORA CHECKLIST — запуск у себя на ПК (Windows + PyCharm)

СОДЕРЖИМОЕ ПРОЕКТА
- backend/  — FastAPI сервер
- frontend/ — красивая страница (v5) — откроется прямо на /

После запуска сервер сам раздаёт:
- Главную страницу:      http://<адрес>:8000/
- API для UI v5:         http://<адрес>:8000/api/v5/calculate
- Старый простой API:    http://<адрес>:8000/calculate


====================
ЧАСТЬ A. Запуск на своём компе (локально)
====================

A1) Открой проект
1) PyCharm → File → Open…
2) Выбери папку проекта (там где backend/ и frontend/).

A2) Установи зависимости (это НЕ установка питона)
В PyCharm снизу открой вкладку Terminal и выполни из корня проекта:

    pip install -r requirements.txt

Если у тебя несколько питонов, лучше так:

    python -m pip install -r requirements.txt

A3) Запусти сервер (локально — только на этом ПК)
В терминале из корня проекта:

    python -m uvicorn backend.main:app --reload --host 127.0.0.1 --port 8000

A4) Открой сайт
Открой в браузере:

    http://127.0.0.1:8000/

====================
ЧАСТЬ A5. Где читать как это работает и как править
====================

Если нужно "понять" и "поправить" без боли:

1) docs/АЛГОРИТМЫ_ПО_ПРОСТОМУ.md
   - как считаются пакет/баллы/итог

2) docs/КАК_ПРАВИТЬ_БЕЗ_БОЛИ.md
   - где менять цены, баллы и тексты (без трогания backend)


====================
ЧАСТЬ B. Чтобы сайт открывался на телефоне/ноуте в домашней сети
====================

B1) Запусти сервер "на всех интерфейсах"
Останови сервер (Ctrl+C) и запусти так:

    python -m uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000

B2) Узнай IP твоего ПК в домашней сети
Windows:
- Win+R → cmd → Enter
- команда:

    ipconfig

Ищи строку "IPv4-адрес" — будет что-то вроде 192.168.1.50

B3) Открой сайт с телефона
Телефон должен быть в том же Wi‑Fi.
В браузере телефона:

    http://192.168.1.50:8000/

B4) Если не открывается — проверь брандмауэр Windows
Иногда Windows блокирует входящие.
Самый простой способ:
1) Пуск → "Безопасность Windows"
2) "Брандмауэр и защита сети"
3) "Разрешить работу приложения через брандмауэр"
4) Найди Python / uvicorn (или добавь вручную) и разреши для "Частные сети".


====================
ЧАСТЬ C. Запуск в один клик (батники)
====================

В корне проекта лежат:
- start.bat                 — меню
- start_localhost.bat       — только на этом ПК
- start_lan.bat             — доступно в домашней сети
- start_secure_localhost.bat — то же, но с паролем
- start_secure_lan.bat       — LAN + пароль (рекомендую, если пробрасываешь порт)

Просто двойной клик по нужному .bat


====================
ЧАСТЬ D. Доступ из интернета (ВАЖНО: безопасность)
====================

Снаружи лучше заходить через VPN на Keenetic (WireGuard). Это безопаснее, чем проброс портов.
Подробная пошаговая инструкция лежит в файле:

    SECURITY_KEENETIC_ДОСТУП_ИЗ_ИНТЕРНЕТА.txt

Если всё-таки нужен проброс порта — обязательно:
- НЕ открывай 8000 напрямую наружу без пароля.
- Лучше: VPN или хотя бы пароль/HTTPS через прокси.
