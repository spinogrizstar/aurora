# Регресс-кейсы: менеджер v5 (пресеты/оборудование)

## 1) «Только опт» без оборудования по умолчанию
1. Выбрать пакет «Только опт».
2. Проверить, что блок «Оборудование» скрыт, кассы = 0, сканеры = 0.
3. Убедиться, что услуга «Подключение сканера» не добавлена автоматически (qty = 0).
4. Итог по часам = 7 (по пресету услуг).

## 2) Переключение пакета «Производитель+розница» → «Только опт»
1. Выбрать «Производитель + розница».
2. Добавить кассы/сканеры, изменить qty услуг вручную.
3. Переключиться на «Только опт».
4. Проверить, что:
   - Все кассы/сканеры сброшены до 0.
   - Услуги пересчитаны строго по оптовому пресету (без мусора от розницы).
   - Все manualOverride/ручные правки сброшены.

## 3) Оборудование в опте включается только вручную
1. На пакете «Только опт» нажать «Показать оборудование (если есть)».
2. Убедиться, что блок оборудования появился и счётчики доступны.
3. Добавить сканеры/кассы — автозаполнение услуг работает только после этого.
