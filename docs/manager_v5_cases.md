# Регресс‑кейсы: менеджер v5 (услуги)

## Пакеты и стартовые часы
1. Выбрать пакет **«Только розница»** → стартовые часы = **9 ч**.
2. Выбрать пакет **«Только опт»** → стартовые часы = **7 ч**.
3. Выбрать пакет **«Только производитель/импортёр»** → стартовые часы = **12 ч**.
4. Выбрать пакет **«Производитель + розница»** → стартовые часы = **18 ч**.

## Кассы и авто‑qty в услугах
1. В детализации услуг увеличить количество касс (ККТ).
2. Проверить, что auto‑qty зависимых услуг растёт **только если не было ручного override**.

## Сканеры меньше касс
1. Установить кассы = 3, сканеры = 2.
2. Проверить, что услуга **«Подключение сканера»** имеет qty = **2**.
