# Регресс-кейсы: менеджер v5 (матрица услуг)

1) retail_only, детализация ON, qty по умолчанию → всего 13ч, 64 350 ₽.
2) wholesale_only, детализация ON → всего 7ч, 34 650 ₽, ККТ блок не показывается.
3) producer_only, детализация ON → всего 12ч, 59 400 ₽, присутствует «настройка принтера 3ч».
4) producer_retail, детализация ON → всего 18ч, 89 100 ₽.
5) Переключение пакетов туда-сюда не оставляет «хвосты» услуг от прошлого пакета.
