# Регресс-кейсы для менеджера v5

Базовая формула менеджера:

```
Итого ₽ = (часы пакета + часы допов + подготовка кассы) × 4 950 ₽
```

## Кейсы

**A)** `retail_only`, ККТ: 0/0/0, допы: 0
- Ожидаемо: 9 ч / 44 550 ₽

**B)** `retail_only`, ККТ: обычные = 2, смарт = 0, другие = 0, допы: 0
- Подготовка кассы: 2×2ч = 4 ч
- Ожидаемо: 9 + 4 = 13 ч / 64 350 ₽

**C)** `producer_only`, ККТ: обычные = 0, смарт = 1, другие = 0, допы: 0
- Подготовка кассы: 1×3ч = 3 ч
- Ожидаемо: 12 + 3 = 15 ч / 74 250 ₽

**D)** `producer_retail`, ККТ: обычные = 0, смарт = 3, другие = 0, допы: все 3
- Допы: 1 + 3 + 3 = 7 ч
- Подготовка кассы: 3×3ч = 9 ч
- Ожидаемо: 18 + 7 + 9 = 34 ч / 168 300 ₽

**E)** `wholesale_only`, ККТ: 0/0/0, допы: интеграция
- Допы: +3 ч
- Ожидаемо: 7 + 3 = 10 ч / 49 500 ₽

**F)** `producer_retail`, чекбокс “Нестандарт/интеграции” выключен
- Ожидаемо: пресейл-баннер **НЕ** показывается.

**G)** `producer_retail`, чекбокс “Нестандарт/интеграции” включен
- Ожидаемо: пресейл-баннер показывается.
