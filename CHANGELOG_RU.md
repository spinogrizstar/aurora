# Журнал изменений

## 2026-02-02 — 0.0.10
- Менеджер v5 переведён на модель «пакет → пресет услуг»: расчёт идёт только по услугам (qty × часы), количество услуг редактируемое, добавлен режим детализации.
- Кассы/сканеры теперь влияют на auto‑qty отдельных услуг без перезаписи ручных значений; добавлен блок «Оборудование» с отдельным счётчиком сканеров.
- Убраны лишние блоки: ТСД (доп.), несколько юрлиц, сценарии (кроме маркера «Нестандарт/интеграции») и поддержка.

## 2026-02-02 — 0.0.9
- ККТ переведён на 3 счётчика (смесь типов), добавлены подсказки, расчёт подготовки кассы по типам, сценарии помечены как маркеры без влияния на цену, пресейл-баннер строго по нестандарту, убраны баллы из UI, добавлены регресс-кейсы.

## 2026-02-02 — 0.0.8

## 2026-02-02 — 0.0.7
